Send soundboard to two devices so you and people in chat both hear it.

Set your game/Discord mic input to the same device you pick as “What others hear” (e.g. SteelSeries Sonar – Chat).

SteelSeries Sonar – use in game chat

Route SoundBoard audio through Sonar so others hear it in voice chat:

  1. Open SteelSeries GG → Sonar mixer.
  2. In the output panel, find SoundBoard (or Electron when running from dev). Drag it onto the Chat or Media channel.
  3. Or use Windows: right‑click speaker icon → Sound settings → Advanced → App volume and device preferences → set SoundBoard’s output to SteelSeries Sonar – Game or Media.

Sonar will mix SoundBoard into your Chat output, so your mic + soundboard are heard together in game.
